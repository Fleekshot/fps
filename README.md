# Minimal Multiplayer Platformer

This project is a barebones 2D multiplayer platformer intended to run in GitHub Codespaces. It uses Node.js, Express and Socket.IO.

## Setup

1. **Install dependencies** (skip if already installed)
   ```bash
   npm install
   ```
2. **Run the server**
   ```bash
   node server.js
   ```
3. Open the forwarded port or navigate to `http://localhost:3000` in the Codespace.

Use the arrow keys to move, `B` to place blocks, right click to delete blocks, `R` to place climbable vines and `Q` to shoot.
