# Browser FPS Example

This project contains a simple browser-based first-person shooter inspired by classic DOOM-style gameplay. It demonstrates basic mechanics such as movement, shooting, enemies, pickups and simple textures generated at runtime.

## Running Locally

1. Install a simple HTTP server (e.g. `python3` built-in).
2. From this directory run:

```bash
python3 -m http.server 8000
```

3. Open `http://localhost:8000` in a modern browser (Chrome/Firefox) to play.

No audio assets are included. Add your own sounds if desired.
